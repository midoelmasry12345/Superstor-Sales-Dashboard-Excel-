Superstore Sales Dashboard (Excel)
Overview

This project presents an interactive Sales Dashboard built in Microsoft Excel using the Superstore dataset. The dashboard provides a clear view of business performance by analyzing key metrics such as sales, profit, and product category trends.

It is designed to help stakeholders quickly explore performance insights and make data-driven decisions through dynamic visualizations and filters.

Objectives

Analyze overall sales and profitability

Identify top-performing categories and regions

Track trends across time

Provide an interactive dashboard for business insights
